# Noticies_github
Aquesta pagina web es divideix en diferents parts
- Index.html
- News.html
- New.html
- Contact.html

## Index.html
En aquest html hi es el codi del home de la pagina web, el home es la carta de presentacio de la pagina web, desde aquesta part pots accedir a les diferents parts de la pagina web

## News.html
En aquest html es el codi on s'emmagatzemen totes les noticies que s'en van publicant dins la pagina web.

## New.html
Dins aquest html el codi tracta de la noticia completa, amb els titols, subtitols i contingut 

## Contact.html 
En aquest html dins el codi hi ha un breu formulari que te demana el teu nom, el teu email, l'assumpte del missatge i el missatge, tambe conte una checkbox i un boto de enviar. A mes dins aquest codi hi ha un mapa on pots trobar la ubicacio de Es Liceu.
